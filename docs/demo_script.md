# Demo Script

1. Open Overview and show baseline KPIs with no active events.
2. Click `Run daily plan` and show the normal-mode optimized plan plus the before/after KPI comparison.
3. Open `Scenarios` and run `supplier_delay`.
4. Return to `Overview` and show the pending approval panel, approval reason, and changed action set.
5. Click `Request safer plan` to show a constrained replan with a new decision id.
6. Approve the safer plan and confirm the plan is applied and removed from the pending queue.
7. Open `Decision Log` and point out the rejected original plan plus the approved follow-up decision.
